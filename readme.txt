The Ultimate WordPress Album, Photo Gallery Plugin. If you need to manage large number of images this plugin is for you.

== Description ==

[Live Demo](https://album-gallery.wpapplab.com/)

[youtube https://www.youtube.com/watch?v=tjcwIPW68sY]

My Album Gallery can showcase your images in gallery and album view. If you need to manage large number of images whether you are a photographer, event manager, photo lover, artist with portfolio or want product gallery you can do it with this plugin. Album can be divided by category and subcategory. Plugin has built in Single and Archive page template. You can add Album or category in sidebar for easy navigation for large number of image decoration.

Before using My Album Gallery plugin you must read the [Documentation](https://album-gallery.wpapplab.com/documentation/) or at least watch this [Get Started video](https://mygallery.wpapplab.com/get-started/) . If you uploaded images before My Album Gallery plugin installation then you must [Regenerate Thumbnails](https://wordpress.org/plugins/regenerate-thumbnails/) using this plugin.
